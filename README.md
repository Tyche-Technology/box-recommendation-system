# 박스 사이즈 추천 시스템

tychetech.io 풀필먼트 매니저를 위한 최적 박스 추천 웹 앱입니다.
상품 크기를 입력하면 기존 박스 카탈로그에서 가장 적합한 박스를 자동으로 추천합니다.

## 시작하기

```bash
# 1. 패키지 설치
npm run install:all

# 2. 초기 박스 데이터 생성 (최초 1회)
npm run seed

# 3. 개발 서버 실행
npm run dev
```

실행 후 **http://localhost:5173** 에서 앱을 사용할 수 있습니다.

## 주요 기능

### 박스 추천 (메인)
1. 제품명, 가로/세로/높이(cm), 무게(kg), 수량을 입력하고 **추가** 클릭
2. 여러 제품을 추가하면 합포장 추천도 가능
3. **최적 박스 추천받기** 클릭 → 공간 활용률 기준으로 결과 표시
4. 원하는 박스를 선택한 뒤 **선택 확정** → 주문 이력에 저장

### 박스 카탈로그
- 사용 가능한 박스 목록 관리 (추가/수정/삭제)
- 활성화 토글로 추천 대상 포함/제외 설정

### 주문 이력
- 확정된 추천 이력 조회
- 기간별 필터링, 제품 상세 확인

### 재고 분석
- 박스별 사용 빈도 및 평균 공간 활용률 통계
- 주문 패턴 기반 최적 박스 사이즈 제안
